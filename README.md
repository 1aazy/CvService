## Installation
Clone repository form GitHub.
```shell
git clone https://github.com/1aazy/CvService.git
```

**For Windows:** <br />
```shell
.\mvnw.cmd install

java -jar target/CvService-0.0.1-SNAPSHOT.jar
```

**For Linux of MacOS:** <br />
```shell
.\mvnw install

java -jar target/CvService-0.0.1-SNAPSHOT.jar
```
